utils::head<br>
<h2>Return the First or Last Part of an Object</h2><br>
<b><u>Usage:</u></b><br>
<code><br>
     head(x, ...)<br>
     ## Default S3 method:<br>
     head(x, n = 6L, ...)<br>
     ## S3 method for class 'data.frame'<br>
     head(x, n = 6L, ...)<br>
     ## S3 method for class 'matrix'<br>
     head(x, n = 6L, ...)<br>
     ## S3 method for class 'ftable'<br>
     head(x, n = 6L, ...)<br>
     ## S3 method for class 'table'<br>
     head(x, n = 6L, ...)<br>
     ## S3 method for class 'function'<br>
     head(x, n = 6L, ...)<br>
     <br>
     tail(x, ...)<br>
     ## Default S3 method:<br>
     tail(x, n = 6L, ...)<br>
     ## S3 method for class 'data.frame'<br>
     tail(x, n = 6L, ...)<br>
     ## S3 method for class 'matrix'<br>
     tail(x, n = 6L, addrownums = TRUE, ...)<br>
     ## S3 method for class 'ftable'<br>
     tail(x, n = 6L, addrownums = FALSE, ...)<br>
     ## S3 method for class 'table'<br>
     tail(x, n = 6L, addrownums = TRUE, ...)<br>
     ## S3 method for class 'function'<br>
     tail(x, n = 6L, ...)<br>
     </code><br>
<b><u>Description:</u></b><br>
<br>
     Returns the first or last parts of a vector, matrix, table, data<br>
     frame or function.  Since ‘head()’ and ‘tail()’ are generic<br>
     functions, they may also have been extended to other classes.<br>
<br>
<b><u>Details:</u></b><br>
<br>
     For matrices, 2-dim tables and data frames, ‘head()’ (‘tail()’)<br>
     returns the first (last) ‘n’ rows when ‘n > 0’ or all but the last<br>
     (first) ‘n’ rows when ‘n < 0’.  ‘head.matrix()’ and<br>
     ‘tail.matrix()’ are exported.  For functions, the lines of the<br>
     deparsed function are returned as character strings.<br>
<br>
     If a matrix has no row names, then ‘tail()’ will add row names of<br>
     the form ‘"[n,]"’ to the result, so that it looks similar to the<br>
     last lines of ‘x’ when printed.  Setting ‘addrownums = FALSE’<br>
     suppresses this behaviour.<br>
<br>
<b><u>Author(s):</u></b><br>
<br>
     Patrick Burns, improved and corrected by R-Core. Negative argument<br>
     added by Vincent Goulet.<br>
<br>
<b><u>Arguments:</u></b><br>
<br/> <br/><b>       x:</b> an object <br/> <br/><b>       n: a single integer. If positive, size for the resulting object:</b> <br/>          number of elements for a vector (including lists), rows for a <br/>          matrix or data frame or lines for a function. If negative, <br/>          all but the ‘n’ last/first number of elements of ‘x’. <br/> <br/><b>addrownums:</b> if there are no row names, create them from the row <br/>          numbers. <br/> <br/><b>     ...:</b> arguments to be passed to or from other methods. <br/> <br>
<b><u>Value:</u></b><br>
<br>
     An object (usually) like ‘x’ but generally smaller.  For ‘ftable’<br>
     objects ‘x’, a transformed ‘format(x)’.<br>
<br>
<b><u>Examples:</u></b><br>
<code><br>
     head(letters)<br>
     head(letters, n = -6L)<br>
     <br>
     head(freeny.x, n = 10L)<br>
     head(freeny.y)<br>
     <br>
     tail(letters)<br>
     tail(letters, n = -6L)<br>
     <br>
     tail(freeny.x)<br>
     tail(freeny.y)<br>
     <br>
     tail(library)<br>
     <br>
     head(stats::ftable(Titanic))<br>
     </code>
